Mattermost

Now we can create a slash-command in Mattermost to interact with our function. A slash-command allows you to enter a / followed by a command, and optionally some arguments, then an HTTP request will be sent to our function. Login to Mattermost and look for the Integrations menu: Then click Slash Command and Add Slash Command. Fill in the following: Key Values Display Name <TEAM-NAME>-echo Description Echo Bot Command Trigger Word <TEAM-NAME>-echo Request URL http://<TEAM-NAME>-echo. »
